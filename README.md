# TailwindCSS + ReactJS Project

Welcome to my first ever website styled using TailwindCSS in ReactJS!

## Overview

This project is a static website built with ReactJS and styled using TailwindCSS. The primary focus is on learning and applying TailwindCSS for styling within a React application. As such, it does not include dynamic content or advanced React features.

## Features

- Utilizes TailwindCSS for responsive and utility-first CSS styling.
- Demonstrates basic ReactJS component structure.
- Implements a simple, clean, and modern design.

## Installation

To run this project locally, follow these steps:

1. **Clone the repository:**

    ```bash
    git clone https://github.com/AnnonymousBanda/Nike-landing-page
    ```

2. **Navigate to the project directory:**

    ```bash
    cd Nike-landing-page
    ```

3. **Install dependencies:**

    ```bash
    npm install
    ```

## Usage

To start the development server and view the project in your browser, run:

```bash
npm start
```
